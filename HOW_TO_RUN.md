# Exchange Rates Website
# React (vite) & FastAPI

### 🏃‍♂️ How To Run

-   Ensure ports 5173, 8080 are free
-   clone the github repository
-   run the `docker-compose up --build`  command in the root directory of the project
-   go to http://localhost:5173 and enjoy!!!
